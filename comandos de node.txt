PARA EJECUTAR NODE APP.JS EN SEGUNDO PLANO
Para ejecutar tu servidor Node.js en segundo plano en Windows, puedes utilizar el administrador de tareas o una utilidad de terceros como PM2. Te mostraré cómo hacerlo con PM2, que es una herramienta popular para administrar procesos Node.js.

Primero, necesitarás instalar PM2 globalmente en tu sistema. Puedes hacerlo ejecutando el siguiente comando en tu símbolo del sistema (cmd):

Copy code
npm install -g pm2
Una vez que PM2 esté instalado, puedes iniciar tu servidor Node.js utilizando el siguiente comando:

sql
Copy code
pm2 start app.js --name "nombre_de_tu_aplicacion"
Reemplaza "app.js" con el nombre de tu archivo de aplicación si es diferente, y "nombre_de_tu_aplicacion" con el nombre que desees darle a tu aplicación en PM2.

PM2 iniciará tu servidor Node.js en segundo plano y te proporcionará un identificador único para administrar el proceso. Además, puedes ver los registros de tu aplicación ejecutando:

Copy code
pm2 logs nombre_de_tu_aplicacion
Para detener tu aplicación en segundo plano, puedes ejecutar:

arduino
Copy code
pm2 stop nombre_de_tu_aplicacion
O para detener y eliminar el proceso de PM2:

arduino
Copy code
pm2 delete nombre_de_tu_aplicacion
PM2 también ofrece otras características útiles, como la capacidad de administrar múltiples aplicaciones, monitorear el estado de la aplicación y reiniciar automáticamente en caso de fallo. Puedes consultar la documentación de PM2 para obtener más información sobre cómo utilizar esta herramienta.
